# Todo-app practice

## Requirement

#### Techs:

- Spring Boot
- RESTful API
- MySQL
- Docker
- Rancher
- JWT
- Jenkins CI

#### Time: 5 days

#### Requirement:

- Create APIs for todo-list app
- Features
-- Register, Login & Logout
-- Create a todo
-- Delete a todo or list todos
-- List all todos
-- Unit test for all APIs

## Run dev environment

- cd todo-app
- run: docker-compose -f compose/dev.yml up
